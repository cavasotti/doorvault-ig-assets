# DoorVault Instagram launch — public image assets

Public host for the @doorvault.app launch slides so the Instagram Graph API can fetch
them by URL. One folder per post (`NN-YYYY-MM-DD-slot-slug/slide-NN.png`). Generated by
`social-posts/launch/build_launch.py` in the (private) DoorVault-Marketing repo. These
are marketing images destined for public Instagram; nothing sensitive lives here.

Image URL shape: `https://raw.githubusercontent.com/cavasotti/doorvault-ig-assets/main/<folder>/slide-NN.png`
